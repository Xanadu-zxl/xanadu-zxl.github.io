## 什么是浏览器？
- 作用：
    1. 将网页渲染出来给用户查看
    2. 能够让用户通过浏览器与网页交互

- 浏览器内核是什么？常见的浏览器内核有哪些？
  浏览器内核：浏览器所采用的渲染引擎，渲染引擎决定了浏览器如何显示网页的内容，以及页面格式信息。

- 常见的浏览器内核：
  1. IE（Internet Explorer）内核 Trident
  2. 谷歌（Chrome）内核 WebKit / Blink
  3. 火狐（Firefox）内核 Gecko
  4. Safari内核 WebKit
  5. 欧朋（Opera）内核 Presto
